RESTful API service (JSON)

How to use

  • POST, PUT, DELETE parameters are packaged
  • GET parameters are url-encoded
  • Parameters in () are primary key
  • Parameters in [] are optional
  • All other parameters are required

API URL calls

URL Method Parameters Description
/register POST (email, password) New user registration
/login POST (email, password) User login
/check POST none Api key check
/usermultiworkout POST Create usermultiworkout
/usermultiworkout GET Get usermultiworkout
/usermultiworkout PUT Edit usermultiworkout
/usermultiworkout DELETE Delete usermultiworkout
/usersingleworkout POST Add usersingleworkout to usermultiworkout
/usersingleworkout GET Get usersingleworkout within usermultiworkout
/usersingleworkout PUT Edit usersingleworkout within usermultiworkout
/usersingleworkout DELETE Delete usersingleworkout from usermultiworkout
/userliftgroup POST Add userliftgroup to user singleworkout
/userliftgroup PUT Edit userliftgroup in user singleworkout
/userliftgroup DELETE Delete userliftgroup from user singleworkout
/userlift POST Add userlift to userliftgroup within user singleworkout
/userlift PUT Edit userlift within userliftgroup within user singleworkout
/userlift DELETE Delete userlift from userliftgroup within user singleworkout
/usermaxes POST Add usermaxes
/usermaxes PUT Edit usermaxes
/usermaxes GET Get usermaxes
/lifts GET Get all lifts (including array of muscles)
/muscles GET Get all muscles
